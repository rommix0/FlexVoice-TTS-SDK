Preformance figures of FlexVoice 3.01

Performance measurement of FlexVoice. 
	*A FlexVoice server is started on Windows2000. The OS was simply choosen, 
	due to it's resource meter.


	*A sample text is choosen, which is 61.5 secs long. This text was issued 
	from the client machines in an infinite loop, and measured time taken to 
	the serve to complete the  the request. If the response time is greater 
	than the length of the text, the server reached its maximal usage.


	*In the table below the measurement were done on different CPUs and the maximal 
	number of simultaneous request were recorded into the table. In other 
	words how many simultaneous requests can the specified CPU fulfill in 
	real-time.
