System Requirements
===================
- Pentium 233 MHz MMX or faster processor
- 32 MB of RAM
- 45 MB of free hard drive space

Installation
============
To install the FlexVoice SDK package:
1. Launch the "setup.exe" program. 
2. Follow the instructions and complete the setup with the default options. 
3. Restart the computer if necessary.
