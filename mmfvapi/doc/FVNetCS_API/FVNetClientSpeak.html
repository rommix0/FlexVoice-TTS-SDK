Creates speech through the callback function.

Documentation

Creates speech through the callback function. This method send the text to the server, the server synthesizes the speech and sends the generated data back to the client. The here given callback function is called periodically to give back the speech data.

Parameters:
handle - The handle created by the FVNetClientCreate method
text - The text to be synthesized
language - The language (see FVNetCSLanguage)
callback - The callback function (FVNetClientCallback) that recieves the data.
param - Any user-specific data that is passed to the callback.
Returns:
The return value is an error code (see FVNetClientErrorCode)
