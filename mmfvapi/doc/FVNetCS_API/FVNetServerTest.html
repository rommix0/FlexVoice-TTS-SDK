Tutorial for FVNetServerTest
Sample program for creating a FlexVoice server

The FVNetServerTest creates a server which will accept text files passed to it by one or more FVNetClientTests return them to there originating client after converting them to wave format.

Usage: FVNetServerTest [options] <port> <channels> <language list>

Where: 

	-sf {8|11|16|22}    Sampling frequency: 8, 11, 16 or 22 kHz. Default: 16
        -br {8|16}          Bit resolution: 8, 16. Default: 16
        -pcm_signed         Specifies the otuput format as signed PCM
        -pcm_unsigned       Specifies the otuput format as unsigned PCM
                            The default output format is signed PCM for 16 bit
                            output and unsigned PCM for 8 bit output.
        -ULAW               Specifies the otuput format as ULAW
        -ALAW               Specifies the otuput format as ALAW
        -VOX                Specifies the otuput format as VOX
        -VIS                Specifies the otuput format as VIS
        <port>              the port where the server should listen
        <channels>          the max count of the channes.
        <language list>     space separated language identifiers (eng hun malay)
                            at least one language should be specified

This simple demo application requires a port number and the required number of the FlexVoice engines to be initialized. As you can see above, the program receives the numbers from the argument.

handle = FVNetServerCreate(); Creates the server which is then initialized, and a handle is issued to the server. If the handle is zero, the initialization was unsuccessful. After successfull server creation the language modules are loaded too in accordance with the language specified by the argument(s).
FVNetServerLoadLanguage( handle, lang )

	FVNetServerInitialize(handle, atoi(atoi(argv[2]), atoi(argv[1])))); a FlexVoice engine must be installed, if not then a FVNetServerError_CannotInitEngine error is generated. The function needs the port number and required number of instances. Two errors are enumerated:\\FVNetServerError_CannotInitEngine
	FVNetServerError_CannotOpenPort

The program waits for the ENTER to be pressed as an event. In case this is true the program closes all FlexVoice Server instances.
FVNetServerDestroy(handle)
