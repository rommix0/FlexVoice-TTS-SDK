Documentation

Initializes the server.
Parameters:
handle - The handle created by the FVNetServerCreate
path - The path where FlexVoice's datas can be found
port - The port where the server should listen
instances - The maximum number of instances that the server should handle This value depends of the computer's power, the best was is to measure it.
format - The required output format (see FVNetCSOutputFormat)
sampleFreq - The sampling frequency
bitResolution - The bit resolution (mainly for PCM output)
Returns:
The return value is an error code (see FVNetClientErrorCode)
