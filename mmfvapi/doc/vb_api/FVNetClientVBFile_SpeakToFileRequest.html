Creates speech.

Documentation

Creates speech. This method sends the text to the server. The server synthesizes the speech and sends back the generated data to the client. The client stores this data in the speech data buffer. Using the FVNetClientVBFile_GetSpeechData method, the caller is able to read out speech data from the speech data buffer. The method doesn't wait for the end of the speech generation it returns immediately. To wait its end you can call the FVNetClientVBFile_Wait method.

Parameters:
handle - The handle created by the FVNetClientVBFile_Create method
text - The text to be synthesized
language - The language (see FVNetCSLanguage)
filename - The name of the output file
speaker - The name of requested speaker. If NULL or empty, then there is no requested speaker.
relVolume - The relative volume (in percent)
relSpeed - The relative speed (in percent)
relPitch - The relative pitch (in percent)
writeSpeed - The speed of the wave generation (in percent)
Returns:
The return value is an error code (see FVNetClientVBFile_ErrorCode)
