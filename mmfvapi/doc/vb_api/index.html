Visual Basic File Client API

Tutorial
Simple program for demostrating a client, running under VB, that connects to a FlexVoice Server.
Reference Manual
Interfaces, classes, enumerations ...
FlexVoice C++ SDK contains an interface for creating a client using Visual Basic 6.0. In the package there is a "FVNetClient_VBFile.dll" file containing the interface functions. To use it, a working server should be reachable.

First a client object should be created, then the client should connect to the server. After these the text data can be sent to the server and the server returns a wave data, that is saved by the client. The client should be closed if it is no longer in use.
