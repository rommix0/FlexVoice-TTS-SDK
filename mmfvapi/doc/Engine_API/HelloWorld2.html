In this example, a class is set up that is doing most of the work. Compared to the previous example the functionality is not extended.
In this example we use two classes: SpeechParams and WavFileWriter.

SpeechParams sp; sp.speaker = "JulieCalm"; sp.samplingFreq = 16000;

With the SpeechParams class we can set the speaking voice("speaker") and various speech parameters("samplingFreq", 'speechRate", "pitch", "pitchRate"). The default value for all of the speech parameters is 0 (zero). In this sample we set the speaking voice to JulieCalm and the sampling frequency to 16000.

The WavFileWriter class is responsible for all operations required to generate speech and save it to a wave file. For details of what operations are needed for speech-generation see the HelloWorld1 example.

WavFileWriter wfw( "Hello World", "hello.wav", sp )

This class constructor requires 3 parameters: the text that will be converted to speech, the output file name and a SpeechParams instance. In our case we save the "Hello world" text into hello.wav file.

if (wfw.writeFile()) return 0; return -1;

At the end of the sample we have a verification that the save to file succeded.
