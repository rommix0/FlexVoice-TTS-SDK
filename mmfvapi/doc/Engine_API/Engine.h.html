Engine.
A TTS Engine is used to produce speech from text. Common operations are:
  • create an engine factory (factories are used for speeding up the initialization of engines, see EngineFactory),
  • create a notification object (typically a Notifier instance) in order to receive notifications about speak-events,
  • create an output site object to be used with the engine (see IOutputSite for the interface; WaveFileOutputSite is a fully implemented class that is capable of writing output to files),
  • pass the notification object to the destination object,
  • create a new engine instance (see EngineFactory and Engine),
  • pass it the destination object,
  • send requests to the engine,
  • manipulate the engine state, in particular the properties of the speaker
  • process notification messages arriving as the engine speaks.
