User Dictionaries

FlexVoice can handle user dictionaries that can modify the standard pronounciation of some specific words. These dictionaries can be loaded from file and can be given to the engines can be enabled or disabled. The dictionaries are stored by the Factory and it can be used by handles (see DictionaryHandle). The handle can provide container functionalities and can give iterators (see DictIterator, ConstDictIterator) to handle the dictionary word by word. The iterators can give an IDictionaryItem that is derived by the IAttribute interface. Through this interface the field of the dictionary entries can be read and modified using that parameter names. For details see IDictionaryItem
