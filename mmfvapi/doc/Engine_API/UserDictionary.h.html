DictionaryHandle
This dictionary implementation is compatible with FlexVoice 10 implementations.
A DictionaryHandle is a list of word-pronunciation associations. A DictionaryHandle instance can be given to the Engine by calling Engine::setDictionary(). The engine will always look up word-pronunciations first in the DictionaryHandle instance, providing an easy way to define the pronunciation of irregular worlds not yet known by the engine.

The DictionaryHandle is implemented with an std::map-like interface. It holds IDictionaryItems that can be retrieved, replaced, etc.
