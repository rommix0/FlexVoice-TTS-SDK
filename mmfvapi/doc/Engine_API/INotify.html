Defines notification methods to be implemented by the API client
virtual void bookmark( const Bookmark& bookmark ) = 0
This method is called when a bookmark is bypassed. Note that by definition processing a bookmark (a notification) should be done in a very short amount of time (no blocking is allowed).

virtual void prepare( const Bookmark& bookmark ) = 0
Should be used as a "pre-notification". By intention, this method is called by an output site just before the bookmark is passed to the actual output device; this should enable the listeners to prepare what they need to prepare. Note that, by definition, processing a bookmark (a notification) should be done in a very short amount of time (no blocking is allowed).
