Tutorials

Sample programs demonstrating of using FlexVoice Engine API
HelloWorld0
This example shows how to instantiate a FlexVoice 3.0 engine and let it speak to a file. Demonstrates the minimal set of calls to use FlexVoice 3.0 in an application.
HelloWorld1
This example is an extention of the previous one in as much as it finds the data directory on your system and also it loads the speaker definitions included in the voicelist.tvl file that acts as a registry of the available voices. This way, the user can change speakers using embedded commands without blocking the output of the speech.
HelloWorld2
In this example, a class is set up that is doing most of the work. Compared to the previous example the functionality is not extended.
Say
This is a useful application that can be used to produce wave files using the text to speech engine. Compared to the previous HelloWorld applications it adds command line handling.
These tutorials show examples of how to use FlexVoice 3.01 in your applications. For Win32 systems all the tutorials have an associated project file for Microsoft Visual C++ 6.0 SP4 that work with a standard installation when the tutorials are on the same driver as the FlexVoice data files and dll. For Linux appropriate makefiles are included. You might also be interested in Compilation Instructions.

Embedded commands

One way to control the engine is to include embedded commands in the text that is sent to the engine. The document here explains the available commands.
