Say

This is a useful application that can be used to produce wave files using the text to speech engine. Compared to the previous HelloWorld applications it adds command line handling.
Creates wave file or annotated phoneme string from given text

Usage:
say -lang language [options] { text | -f input_file } [output_file]

Options: 

    -phout              Create phoneme string output
    -speaker name       Specifies the speaker
    -sf {8|11|16|22}    Sampling frequency: 8, 11, 16 or 22 kHz. Default: 16
    -br {8|16}          Bit resolution: 8, 16. Default: 16
    -raw                Specifies the file format as raw (without header)
    -pcm_signed         Specifies the otuput format as signed PCM
    -pcm_unsigned       Specifies the otuput format as unsigned PCM
    -ulaw               Specifies the otuput format as ULAW
    -alaw               Specifies the otuput format as ALAW
    -vox                Specifies the otuput format as VOX
    -vis                Specifies the otuput format as VIS
                        The default output format is signed PCM for 16 bit
                        output and unsigned PCM for 8 bit output.
    -speechrate value   Speech rate in percents
    -pitch value        Default pitch of speech in Hz
    -pitchrate value    Pitch rate in percents

  language: one of "eng", "hun", or "malay"

  text: a single word or text in quotes

  If output_file is not specified, it defaults to sound.wav
