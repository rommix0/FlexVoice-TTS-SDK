Output Sites.

IOutputSite
Interface called by the TTS engine to send out data when it becomes available.
IWaveOutputSite
Interface called by the TTS engine to send out wave data when it becomes available.
IPhonemeOutputSite
Interface called by the TTS engine to send out phoneme data (in the form of phoneme codes) when it becomes available
IAnnPhonemeOutputSite
Interface called by the TTS engine to send out annotated phoneme data (using phoneme codes) when it becomes available
IAnnIPAPhonemeOutputSite
Interface called by the TTS engine to send out annotated IPA phoneme data when it becomes available
