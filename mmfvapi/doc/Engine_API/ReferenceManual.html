Conventions

Ownership of objects is non-trivial issue in C++. In designing this interface we have used the convention that when an object is passed as a pointer then the receiver takes over the ownership of the object. Also, if an object is passed by reference or value, the ownership is not taken (i.e., no delete will be called on the object, etc.).

Demo application

The source code of the Say demo application can be found in the directory Source. This shows how to create and initialize and engine, how to send data to it, and how to save the output as a wave-file. The demo also shows examples of how to control some of the many controllable parameters of the TTS engine.
