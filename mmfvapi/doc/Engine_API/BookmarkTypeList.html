Documentation

A BookmarkTypeList provides a comfortable interface to be used with INotifyDispatcher methods, namely with registerNotify, addBookmarkTypes and removeBookmarkTypes. Note that you can call e.g. registerNotify with the syntax registerNotify( aNotifyObject, BM_STOP | BM_PAUSE ).

See also operator|.
