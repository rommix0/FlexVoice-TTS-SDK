Phoneme.h

IPAPhoneme
Describes phonemes by their features, essentially providing a language-independent representation of phonemes.
IPhonemeCoder
Interface to convert between phoneme symbol tables and the above IPA phoneme representations.
PitchContour
A vector of timepoint-pitch pairs; time is relative to the phoneme duration and pitch is relative to the speaker's default pitch
LoudnessContour
A vector of timepoint-loudness pairs; time is relative to the phoneme duration and loudness is relative to the speaker's default loudness
AnnIPAPhoneme
This struct contains an IPAPhoneme object and prosodic information
