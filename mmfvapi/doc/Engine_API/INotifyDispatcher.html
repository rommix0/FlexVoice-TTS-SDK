Documentation

Interface for managing notification dispatching. An INotifyDispatcher interface stores a number of INotify instances. When a notification arrives, it goes through this internal list and calls the notification method of all the INotify instances for which the given notification type is permitted. An INotifyDispatcher typically implements an INotify interface, as well.
virtual void registerNotify( INotify* notify, const BookmarkTypeList& types ) = 0
Adds the given INotify instance to the internal list of INotify instances managed by the dispatcher. Ownership is not transferred. The client is responsible for managing the lifetime of the INotify instance.
Parameters:
notify - non-zero pointer to an INotify instance.
types - `list' of bookmark types, the given INotify instance will be notified of.
