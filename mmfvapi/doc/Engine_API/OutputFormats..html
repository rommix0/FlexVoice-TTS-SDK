Output Formats.

IOutputFormat
Abstract base class for all TTS output formats.
WaveOutputFormat
IOutputSite instances having a WaveOutputFormat inform the engine that they expect wave input from the engine.
PhonemeOutputFormat
IOutputSite instances having a PhonemeOutputFormat inform the engine that they expect phoneme input from the engine.
AnnPhonemeOutputFormat
IOutputSite instances having a AnnPhonemeOutputFormat inform the engine that they expect phoneme input from the engine.
AnnIPAPhonemeOutputFormat
IOutputSite instances having a AnnIPAPhonemeOutputFormat inform the engine that they expect phoneme input from the engine.
TextOutputFormat
NOT YET IMPLEMENTED.
A TTS engine can produce either normalized text output, phoneme output with prosody information, or speech (wave).

Upon construction an IOutputSite instance is passed to the TTS Engine (see createEngine). In order to determine the type of output expected, the TTS Engine calls the getOutputFormat method of the passed IOutputSite instance, which in turn returns an IOutputFormat instance. The engine then sets itself up so that it will be able to support the given IOutputSite instance.
