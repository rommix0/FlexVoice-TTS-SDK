( const char* text,

Creates speech through the callback function.

Documentation

Creates speech through the callback function. The here given callback function is called periodically to give back the speech data. (see FVServerCallback)

Parameters:
text - The text to be synthesized
language - The language of the given text
format - The required output format (see FVServerOutputFormat)
sampleFreq - The sampling frequency
bitResolution - The bit resolution (mainly for PCM output)
callback - The callback function (FVServerCallback) that recieves the data.
param - Any user-specific data that is passed to the callback.
Returns:
The return value is an error code (see FVServerErrorCode)
