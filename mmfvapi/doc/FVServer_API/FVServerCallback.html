Documentation

The type of the callback function, This function is called for passing the data from the server to the client.
Parameters:
param - The 'param' parameter of the FVServerSpeak method.
data - The output data-chunk. Its format depends on the 'format' parameter of the FVServerSpeak method.
size - The size of the data-chunk in bytes.
Returns:
The return value indicates if the speak is to stop or not. In case of returning with 0 the speak will be stopped.
