Documentation

The possible output formats
FVServerOutputFormat_Invalid
Invalid output format. Used to indicate that the format hasn't been set yet

FVServerOutputFormat_PCM_Signed
Signed PCM format. The output is given as samples (with the given bit resolution) and its value should be treat as signed.

FVServerOutputFormat_PCM_Unsigned
Unsigned PCM format. The output is given as samples (with the given bit resolution) and its value should be treat as unsigned.

FVServerOutputFormat_ULAW
ULAW format. The output data contains the samples in a coded format. The bit resolution should be set to 16 that will be coded to 8 bit.

FVServerOutputFormat_ALAW
ALAW format. The output data contains the samples in a coded format. The bit resolution should be set to 16 that will be coded to 8 bit.

FVServerOutputFormat_VOX
VOX format The output data contains the samples in a coded format. The bit resolution should be set to 16 that will be coded to 8 bit.

FVServerOutputFormat_VIS
VIS format The output data contains the samples in a coded format. The bit resolution should be set to 16
