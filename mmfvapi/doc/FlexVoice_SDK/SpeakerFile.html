Speaker File

Tells the required files for voices distribution
The program will show all the required voice files for the given .tav file. you can find the executable in: <instalation path>\FlexVoice3 SDK\bin directory.

Usage: SpeakerFile.exe SPEAKER.tav

Where:

SPEAKER.tav is the voice's tav file.

Example:

SpeakerFile C:\MyPath\SomeWhere\My_speaker.tav
