Redistribution of voices

Use the SpeakerFiles.exe application to determine which files are required for any given voice.

Usage: SpeakerFiles.exe speakername

The result is a list of files which must be copied to the Data_3_01_000 folder in order that the given voice function correctly assuming that the FlexVoice engine is installled.

After the files are copied to the directory, please edit the [Common Files] Voicelst.tvl file. This is to "register" the voice in the available voices list.

For Example: "MyNewVoice" "Voices"\MYNEW.tav
