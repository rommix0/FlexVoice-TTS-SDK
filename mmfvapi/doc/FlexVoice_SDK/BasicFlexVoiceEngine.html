Basic FlexVoice Engine

The setup of the FlexVoice-engine requires the following steps: - copy the [system files] to the Window's system directory

- copy the [data files] to the data directory in Mindmaker's common files directory:%MINDMAKER_COMMON_FILES%/FlexVoice/Data_3_01_000 (E.g. if %MINDMAKER_COMMON_FILES% is C:/Program Files/Common Files/Mindmaker then this directory will be C:/Program Files/Common Files/FlexVoice/Data_3_01_000

- create a string-type registry-entry to define the Mindmaker's Common directory: HKLM/software/Mindmaker/CommonDirectory = %MINDMAKER_COMMON_FILES% (e.g.: C:\Program Files\Common Files\Mindmaker)

- install at least one speaker using the info from file: RedistribuitionofVoices.html The overlapping files are equal so it is the same whichever one is used

- select a speaker and copy its tav file to the [data directory] as Default.tav

- create a VoiceList.tvl file in the [data directory] listing the speakers you want to use. (Each line should define a speaker: name <TAB> filename. E.g. Julie Voices/Julie.tav)
