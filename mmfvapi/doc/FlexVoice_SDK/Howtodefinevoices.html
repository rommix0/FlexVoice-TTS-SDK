How to define voices?

Voice definitions are to be found in .tav, or voice definition files. The voice definition files that come with FlexVoice can be found in the Voices subdirectory of the FlexVoice data directory. The voicelist.tvl file is normally used by programs to enumerate the voices available on the system, so the voicelist.tvl file acts as a registry. However, using the C++ API you can add individual voices to the engine by just loading the voice definition directly from a .tav file. Or you can even instatiate a Speaker object and fill in the various parameters.
