Installation

To install SDK, you need to have at least 60MB free space on the HDD. SDK itself only uses 35 MB. SDK provides various interfaces. In order to use them the Developer tool is requred to be accessable. For example. JavaSDK, Visual C++ or Visual Basic. After installations you can visit FlexVoice's WebSite for registration or for further information on the product.
