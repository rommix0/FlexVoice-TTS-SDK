FlexVoice Net Client/Server interface

This interface is for applications that uses FlexVoice in Client/Server model with TCP/IP protocol. The following files take the part of the interface
  • SDK/MMFVAPI/include/FVNetCS/FVClient.h
  • SDK/MMFVAPI/include/FVNetCS/FVNetClient.h
  • SDK/MMFVAPI/include/FVNetCS/FVNetServer.h
  • SDK/MMFVAPI/lib/FVNetClient.lib
  • SDK/MMFVAPI/lib/FVNetServer.lib
  • <Windows's system>/FVNetClient.dll
  • <Windows's system>/FVNetServer.dll
