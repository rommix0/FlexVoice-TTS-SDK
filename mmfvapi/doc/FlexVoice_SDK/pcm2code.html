pcm2code

Converting PCM wave to vox or ulaw.
Converting PCM wave to vox or ulaw The program takes an input file in wave file format coded in standard PCM coding and outputs a specially coded -ULAW or VOX- wave file. In order to play the "converted" wave file the approporiate codec should be installed, or a suitable software application, for example CoolEdit, SoundForge etc.

Usage: pcm2code.exe input_file output_file coding_type <ulaw or vox>

Where:

input_file = the PCM coded Wave file output_file = the recoded file using the codec specified in coding_type coding_type = the codec required to be used for encoding, eg. ULAW or VOX

Example:

C:\Anypath\pcm2code.exe record.wav coded.wav vox
