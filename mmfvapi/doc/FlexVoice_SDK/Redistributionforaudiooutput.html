Redistribution for audio output

For accessing the audio object the engine uses the AudioMixer module. It contains three self registerable files. The files should be copied to the same directory, register the two dll-s using the regsv32.exe and start the exe for registering it into the system. Of course it is possible to have a different, custom solution for the wave output site. Using AudioMixer is not compulsory. 

	[common files]
	AudioMixerps.dll
	AudioMixerServer3.exe
	AudioMixer3.dll

The basic engine and at least one voice is required for FlexVoice TTS to work. Basic Engine Redistribution
