FlexVoice File Client for Visual Basic

A simple Visual Basic interface to access the FlexVoice Server application. It can connect to a server using Tcp/IP protcol, save the speech into various wave file formats using different voices with wide range of possibility to change the voice paramaters. The application itself could be located in <installation path>:\FlexVoice3 SDK\bin
