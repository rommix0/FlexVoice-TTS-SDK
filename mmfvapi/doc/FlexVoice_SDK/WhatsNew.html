What's New?

Version 3.01
Mainly bugfixes and no bigger changes externally. Only the inner parts were transformed slightly to make it possible to implement new language(s).
  • multi-language module (only important in multi-language environment)
  • bugfixes (pitch fluttering removed...)
  • new embedded commands


Version 3.0
This is the first official release of FlexVoice 3.0. Compared to FlexVoice 2.0 the following changes have been implemented:

  • better pronounciation model
  • enhanced phrase boundary detection model
  • enhanced word prominence detection model
  • improved duration model
  • improved intonation model
  • new loudness contour model
  • bugfixes
  • tuned voices
  • multiple language support
