Redistribution for wave file saving

Wave file saving is supported through the engine's Wave output site. It supplies a data stream which could be redirected into a file. There is a sample on this solution using the WaveFileWriter.cpp in <instalation path>\FlexVoice3 SDK\MMFVAPI\samples\.
